Installation instructions
#######################

Install using conda and pip
----------------------------

.. code-block:: console
    
    conda create -n biotuner python=3.8
    conda activate biotuner
    pip install biotuner

Install from source
-------------------

.. code-block:: console

    git clone git@github.com:AntoineBellemare/biotuner.git
    cd biotuner
    pip install -e .
